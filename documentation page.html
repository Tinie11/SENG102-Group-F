ABOUT OUR FRAMEWORK

  • Our CSS based framework is a simple library that provides basic templates and designs of different tags and elements which allow for easy and fast web designing
  • it is easy to use and understand, even for beginners

Our framework is to be used primarily for the designing of educational based websites and webpages

}
