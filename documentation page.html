ABOUT OUR FRAMEWORK

  • Our CSS based framework is a simple library that provides basic templates and designs of different tags and elements which allow for easy and fast web designing
  • It is easy to use and understand, even for beginners

    • Our framework is to be used primarily for the designing of educational based websites and webpages

    /*by default*/ body{ background-color: white;
    color: #000;
    letter-spacing: normal;
    line-height: 1.5px;
    text-align: justify;
    margin: 0;
    font: 2em sans-serif;
    }
    /*styling headings*/
    h1,h2,h3,h4,h5,h6{
    margin: 50px;
    font-weight: 400;
    font-style: bold ;
    text-align: center;
    }
    Above is a snippet of our css stylesheet showing some of the default stylings.
  • To start using our framework, add the CSS link to your html document and include the classes or ids to your tags

    • <link rel="stylesheet" href="SENG PROJECT CSS FRAMEWORK.css">

  • Different elements of the html file such as body, header, etc. have been styled by default.
  • Class names for different elements and tags are easy to remember and identify.
    • .hr1{
    width:50%;

    margin: none;
